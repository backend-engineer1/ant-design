# 介绍

- category: start
- order: 0

---

下一个设计&前端框架，来自蚂蚁金服体验技术部的一套全新的企业级前端开发框架。

![](https://t.alipayobjects.com/images/rmsweb/T11aVgXc4eXXXXXXXX.svg)

## 特性

- 丰富实用的 React UI 组件。
- 基于 React 的组件化开发模式。
- 基于 webpack 的调试构建方案，支持 ES6。
- 背靠 npm 生态圈。
- 提炼于企业级金融产品的交互语言和视觉体系，使用 Sketch3 设计。
