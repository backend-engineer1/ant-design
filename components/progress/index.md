# Progress

- category: Components
- chinese: 进度条

---

## 使用

`<Line perscent="30" />`

`<Circle perscent="30" />`

## 属性参数

| 参数     | 说明           | 类型     | 默认值       |
|----------|----------------|----------|--------------|
| perscent    | 百分比           | number   | 0           |
| status   | 状态，有两个值normal、exception | string   | normal |
| strokeWidth | 进度条线宽度，单位是进度条画布宽度的百分比 | number | 1           |
| width | 进度条画布宽度，这里没有提供height属性设置，Line型高度就是strokeWidth，Circle型高度等于width | string | null |
