# Tree

- category: Components
- chinese: 树形控件

---

## 何时使用

简单的树形菜单应用，暂不支持键盘操作。

## API

| 参数      | 说明                                     | 类型       | 默认值 |
|-----------|------------------------------------------|------------|--------|
| checkable | whether support checked | bool     | false    |
| expandAll     | expand all treeNodes    | bool | false     |
