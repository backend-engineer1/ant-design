# Card

- category: Components
- chinese: 卡片
- type: 展示

---

通用卡片容器。

## 何时使用

最基础的卡片容器，可承载文字、列表、图片、段落，常用于后台概览页面。

```html
<Card title="卡片标题">卡片内容</Card>
```

## API

### Select props

| 参数     | 说明           | 类型     | 默认值       |
|----------|----------------|----------|--------------|
| title    | 卡片标题 | React.Element   |  无  |
