## api

是个方法，参数只有一个 object ，具体成员如下

| 成员     | 说明           | 类型     | 默认值       |
|----------|----------------|----------|--------------|
| title    | 标题           | React.Element   | 无           |
| content   | 面板内容 | React.Element   | 无 |
| onOk     | 确定回调 | function | 无           |
| onCancel | 取消回调 | function | 无           |
| width | 宽度 | String or Number | 无           |
