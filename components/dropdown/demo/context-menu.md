## zh-CN

默认是移入触发菜单，可以点击鼠标右键触发。

## en-US

The default trigger mode is `hover`, you can change it to `contextMenu`.

```css
.site-dropdown-context-menu {
  color: #777;
  background: #f7f7f7;
}
```
