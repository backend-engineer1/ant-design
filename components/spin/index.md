# Spin

- category: Components
- chinese: 加载动画
- type: 展示

---

用于页面和区块的loading状态。

## 何时使用

页面局部处于等待数据渲染时。

## API

| 参数       | 类型            | 默认值       |说明           |
|-----------|----------------|-------------|--------------|
| size        | enum      | default       | spin组件中点的大小，可选值为small default large
